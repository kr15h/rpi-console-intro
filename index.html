RPi Console Intro

Most used Linux console commands for operating on the Raspperry Pi.

This is a list of frequently used console commands while using Raspberry Pi with Raspbian. This document is a combination of knowledge gathered from online materials and experience. Feel free to contribute via GitHub.

This is important once you start to dig deeper in programming. It does not matter what kind of programming as the console is used a lot as in web, app and other kind of development today. This was made to support a couple of Raspberry Pi workshops.

The commands

ls

Lists curent directory contents. By issuing this command you can see what files and folders there are in the current directory you are.

pwd

Tells the current directory you are in starting from the root level. For example:

/home/pi/user

cd

Change directory. This is the main navigation command
